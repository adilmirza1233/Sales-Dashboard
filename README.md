# 📊 Superstore Sales Dashboard

An interactive Sales Analytics Dashboard built with Python, Pandas, Plotly, and Streamlit.

## 🔍 Features
- Filter by Region, Category, Sub-Category
- KPI Cards — Total Sales, Profit, Orders, Profit Margin
- Sales & Profit by Category
- Monthly Sales Trend
- Top 10 Customers & Products
- Sub-Category wise Profit Analysis

## 🛠️ Tech Stack
- Python
- Pandas
- Plotly
- Streamlit

## 📁 Dataset
- Superstore Sales Dataset (Kaggle)
- 9994 rows, 21 columns

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
